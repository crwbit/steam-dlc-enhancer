# Steam DLC Enhancer

Steam's DLC lists can get messy, especially for games with hundreds of add-ons. This extension adds a compact dashboard to the store page so you can find exactly what you're looking for without the endless scrolling.

## What it does
* **Search & Filter**: Instantly find DLCs by name or filter by "Unowned," "Discounted," or "Free."
* **Cost Calculator**: See exactly how much it costs to complete your collection.
* **Savings Tracker**: Displays how much you're saving during sales across the whole list.
* **Smart Sorting**: Sort by price, name, or the biggest discount.
* **Native Look**: Built to match Steam's design perfectly.

## Installation
1. Download or clone this folder.
2. Open Chrome and go to `chrome://extensions`.
3. Turn on **Developer mode** (top right).
4. Click **Load unpacked** and select this project folder.
