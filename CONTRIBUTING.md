# Contributing to Steam DLC Enhancer

Thanks for wanting to help out! To keep the extension stable and looking clean, please follow these simple guidelines.

### Reporting Issues
If you find a bug or something looks broken:
* **Check the URL**: Tell us which game page had the issue.
* **Describe the problem**: What happened, and what did you expect to happen instead?
* **Language**: Let us know what currency or language your Steam store is set to (this helps us fix pricing bugs).

### Code Changes
If you’re submitting a fix or a new feature:
1. **Match the Style**: Use the same colors and fonts already in the extension. It should always feel like it belongs on Steam.
2. **Keep it Fast**: Don't add heavy libraries. We want the extension to stay lightweight so it doesn't slow down the browser.
3. **Test Local**: Make sure your changes don't break the search or filter functions before sending a pull request.
